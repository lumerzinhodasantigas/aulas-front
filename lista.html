Trabalhando com listas

listas são estruturas para reprensetação de conteúdo na forma de relação de itens.

Lista não-ordenada

Relação de itens em que a sequência não importa.

Exemplos:

  • Teclado
  • Mouse
  • Webcam
  • Monitor

Lista ordenada

Relação de itens em que a sequência importa.

Exemplo:

  1. Andressa
  2. Bruno
  3. Carla
  4. Mônica
  5. Tiago

Listas aninhadas

podemos misturar listas de diferentes tipos e também colocar listas dentro de outras listas

Exemplo:

  • Doces
    • Brigadeiro
    • Beijinho
    • Torta de Limão
    • Torta de Chocolate
  • Salgados
    • Coxinha
    • Kibe
    • Esfiha
    • Croassaint

Lista de Definição

Lista usada para estrutura de itens baseados em termo e descrição. útil para conteúdos tipo: glossários, FAQs, ou apresentação de dados com algum conceito, etc.

Exemplos:

Processador
É a unidade de processamento computacional.
Peça essencial em qualquer equipamento digital.
Memória RAM
Espaço de armazenamento volátil.
